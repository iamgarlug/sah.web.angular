Welcome

My name is Stephen Humburg. This site is a professional portfolio where you can learn about my background, download my resume, and find links to my work and professional profiles.

I built it myself as a hands-on example of how I leverage modern tools to ship software efficiently. The design is intentionally minimal — the priority here is technical implementation over visual polish. I used Claude Code throughout: scaffolding components, implementing routing, writing TypeScript and HTML, and configuring the CI/CD deployment pipeline.
